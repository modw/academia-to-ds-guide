Academia to Data Science: A Minimally Sufficient GuideÂ¶

Welcome to the ultimate guide to prepare for a transition from academia to data science. Unlike your standard online data science educational material, this guide is meant to provide you with pointers on how to translate your already existing skills to get industry jobs while filling any gaps if necessary.

This guide will work best if you are a Ph.D. in a natural science field because it assumes some experience in fundamental scientific research. If you have more applied science experience than that, you are better positioned than we were when we started working on our transition.
